Detecte automaticamente containers do InfluxDB rodando no Docker

Como executar o InfluxDB 1.11.8 no Docker:

🪟 Docker no Windows: docker run -d -p 8086:8086 --name influxdb influxdb:1.11.8
🐧 Docker no WSL2: docker run -d -p 8086:8086 --name influxdb influxdb:1.11.8 Nota: Use o IP do WSL2 para acessar (será detectado automaticamente)
📋 Verificar containers: docker ps (ativos) ou docker ps -a (todos)
▶️ Iniciar container parado: docker start influxdb
🛑 Parar container: docker stop influxdb
